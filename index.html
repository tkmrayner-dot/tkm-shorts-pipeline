Private desktop video automation tool

Create, caption, and prepare gaming Shorts.

TKM Shorts Pipeline is a private local workflow used by the app owner to process recorded gaming videos, create vertical clips, generate clip-specific metadata, and upload finished videos to the owner's connected accounts.

How the app works

The app runs locally on the owner's desktop computer. It is not a public website, public login service, or user-generated content platform.

Local video processing

The owner supplies their own recorded gameplay videos. The pipeline creates short clips for review and posting.

Clip metadata

The tool prepares titles, captions, descriptions, and tags for each finished short-form clip.

Account authorization

TikTok Login Kit is used only so the app owner can authorize posting to their own TikTok account.

TikTok integration

The app uses TikTok Login Kit for authorization and TikTok's Content Posting API to upload completed videos. Upload access is only used for the app owner's own account and own video content.

  • Uses owner authorization through TikTok Login Kit.
  • Uploads owner-created gaming clips through the Content Posting API.
  • Does not provide public upload access to other users.
  • Does not collect data from unrelated TikTok users.

Private use only

TKM Shorts Pipeline is an internal/private creator tool. It has no public signup flow, no user dashboard, and no paid customer accounts. The website exists to describe the app and provide policy documents.

  • No public accounts
  • No public login page
  • No third-party users
  • No sale or sharing of personal data

Review note

This website is the public information site for the private desktop application. The application itself runs on the owner's local Windows computer and is used only to upload the owner's own finished gaming clips after authorization.

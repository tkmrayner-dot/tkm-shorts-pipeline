What this page is for

This page is used as the redirect URI for TikTok Login Kit authorization. After the app owner authorizes the app with TikTok, TikTok redirects the browser back to this page with an authorization code in the URL.

Private desktop workflow

The returned code is used only by the app owner to complete authorization for the local TKM Shorts Pipeline desktop workflow.
